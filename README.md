# GuardListApp Nedir?

  Bu uygulama 1 aylık periyotta nöbet tutulan yerlerde nöbetlerin daha adil olarak dağatılmasını sağlamak amacıyla geliştirilmiştir.
Uygulama üzerinden kişilerin mazeretli günlerini girerek bu günlerde nöbet tutmasını engelleyebilirsiniz. Atama tuşlarını kullanarak nöbet listesinde değişiklikler yaratabilirsiniz. 
Daha adaletli bir sistem için manuel olarak atama yapamazsınız! Nöbet listesini excel çıktısı olarak alabilirsiniz.
